# minisam

这是一个由图优化（北京）科技有限公司开发的因子图共享软件。

感谢gtsam作者杰出的工作。

## 我们遵循了以下原则来开发：

1. 尽量不用模板.
2. 不用boost.
3. 不用traits.
4. 不用智能指针.

## 依赖库:

1. gcc 编译器版本:>=5.4.0 20160609 (Ubuntu 5.4.0-6ubuntu1~16.04.11)

​      2. Cmake版本>=2.6

​      3. eigen3 

## 安装:

1. 拷贝库至本地.

   git clone https://github.com/shaolinbit/minisam_lib.git

2. 运行以下命令: 

	cd minisam_lib
	mkdir build
	cd build
	cmake ..
	sudo make install
